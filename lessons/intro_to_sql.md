# Intro to Web SQL

## Desired outcomes from this session:

The purpose of this session is to ensure each participant feels comfortable with the basics of SQL. We'll cover:

- [ ] Intro
- [ ] Syntax
- [ ] Query
- [ ] Requests
- [ ] Clauses
- [ ] Using the [SQLite DB Browser](https://sqlitebrowser.org/)
